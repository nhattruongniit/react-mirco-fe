# Mirco Frontend Demo

A sample repo for demoing a micro frontend architecture setup.

## Getting started
```bash
# start app-shell: http://localhost:8001
$ cd app-shell
$ npm run install
$ npm run dev

# run remote cart-app: http://localhost:4174
$ cd cart-app
$ npm run install
$ npm run build
$ npm run preview 

# start product-app: http://localhost:4173
$ cd cart-app
$ npm run install
$ npm run build
$ npm run preview
